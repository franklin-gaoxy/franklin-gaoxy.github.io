# 简介
三个功能:

1. 创建用户
2. 检查token(提交token和username即可知道当前用户的状态)
3. 登录(登录后会返回一个状态和token)
# 使用
## 登录
> 提交用户名密码,返回一个状态 true为登录成功 同时返回一个token
